## Video Production Guidelines

### Format
- **Resolution**: 1920x1080 (1080p)
- **Frame Rate**: 60fps for code demos
- **Audio**: Clear voiceover with background music at 10% volume
- **Code Font**: JetBrains Mono, 18pt
- **Terminal Theme**: Dark (Dracula or similar)

### Structure Template
1. **Intro** (30 sec): Title card + brief overview
2. **Concept** (2-3 min): Explain the topic
3. **Demo** (5-10 min): Live coding demonstration
4. **Summary** (1 min): Key takeaways
5. **Next Steps** (30 sec): What to learn next
